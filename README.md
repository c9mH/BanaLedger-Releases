# HouTools Releases

Public release assets and update metadata for HouTools applications.

## Products

- **BanaLedger**: personal finance tracking for Windows and Android.
- **HouKi**: local-first knowledge management for Windows, Linux, Web/PWA, and Android.

## Release Naming

| Product | Release tag | Asset prefix |
| --- | --- | --- |
| BanaLedger | `banaledger-v<version>` | `BanaLedger-v<version>-...` |
| HouKi | `houki-v<version>` | `HouKi-v<version>-...` |

The legacy BanaLedger `v<version>` tags remain supported through the
`BanaLedger-` asset prefix. Each application filters the shared release list by
its own tag and asset prefix.

## Downloads

Download builds from the [Releases](https://github.com/c9mH/HouTools-Releases/releases)
page. Published assets may include Windows builds, Android APKs, portable
archives, and SHA-256 checksum files.

Verify the matching `.sha256` file before running or installing a downloaded
build. Application source repositories may remain private; update checks only
read this public repository and do not require a GitHub token.
